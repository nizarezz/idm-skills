# IDM AI Writing Detox

Good writing is invisible. If readers notice the writing style, it's distracting from the content. AI patterns are noticeable — they break trust, especially in industrial journalism where precision and credibility are paramount.

## When to Use This Skill

- Writing or editing any IDM content in French, Arabic, or English
- Creating press releases or media advisories
- Drafting industrial reports or newsletters
- Writing social media posts for IDM channels
- Reviewing any AI-assisted text before publication

## Core Principle

**If readers think "AI wrote this," you've already lost their trust.**

Industrial readers — executives, engineers, government officials — are especially attuned to vague, inflated language. They deal in specifics: tons, percentages, timelines, regulations. AI slop is the opposite of industrial precision.

## FRANÇAIS — Mots et Expressions Interdits

### Mots Interdits (Supprimer ou Remplacer)

| Mot | Pourquoi c'est mauvais | Alternative |
|-----|----------------------|-------------|
| approfondir | Mot signature IA | examiner, analyser, explorer |
| tapisserie | Prose violet | (supprimer entièrement) |
| paysage | Jargon d'entreprise | situation, environnement |
| levier | Jargon | utiliser |
| robuste | Sans signification | fiable, solide, puissant |
| holistique | Jargon d'entreprise | complet, intégral |
| synergique | Jargon d'entreprise | (supprimer, reformuler) |
| paradigme | Surutilisé | modèle, approche |
| innovant | Éloge vide | (décrire ce qui est nouveau) |
| transformationnel | Hyperbole | modifié, amélioré |
| écosystème | Métaphore surutilisée | système, réseau |
| riche | (comme modificateur) | (supprimer ou être précis) |
| plus de (pour quantité) | Règle AP | davantage de |
| faciliter | Jargon | aider, permettre |
| implémenter | Jargon | mettre en œuvre, créer |
| fonctionnalité | Jargon | fonction, caractéristique |
| méthodologie | Jargon | méthode |

### Expressions Interdites (Supprimer Entièrement)

**Phrases de purgatoire (supprimer):**
- "Il est important de noter que..."
- "Dans le paysage industriel actuel..."
- "Plongeons dans..."
- "Sans plus attendre..."
- "Dans cet article, nous allons..."
- "Il convient de mentionner que..."
- "Comme nous le savons tous..."
- "Dans un premier temps..."

**Hedges vides:**
- "Pour être honnête..."
- "Au final..."
- "En ce qui concerne..."
- "En termes de..."
- "S'agissant de..."

**Enthousiasme IA:**
- "C'est une avancée majeure"
- "...et c'est une bonne chose!"
- "Voilà le point:"
- "...et c'est tant mieux!"

**Jargon d'entreprise:**
- "Dans la durée..."
- "À l'avenir..."
- "À ce stade..."
- "Du fait que..."
- "Afin de..."

### Structures Interdites

**Ne pas commencer des phrases avec:**
- "Alors," ou "Bon,"
- "Maintenant," (quand il ne s'agit pas du temps)
- "Écoutez," ou "Regardez,"
- "Fondamentalement," ou "Essentiellement,"

**Ne pas terminer des phrases avec:**
- "...n'est-ce pas?"
- "...vous savez?"
- "...si on veut"

### Le Mot "Révolutionnaire" dans l'Industrie

**Évitez:** "C'est révolutionnaire pour l'industrie marocaine"
**Évitez:** "Une révolution dans le secteur automobile"
**Dites plutôt:** Décrivez le changement réel sans hyperbole

## العربية — كلمات وتعبيرات محظورة

### كلمات محظورة (احذف أو استبدل)

| الكلمة | لماذا سيئة | البديل |
|--------|-----------|--------|
| يهدف إلى (في سياق مبهم) | نمط الذكاء الاصطناعي | يهدف تحديداً إلى / يعمل على |
| شامل | ج的语言 الشركة | كامل / متكامل |
| مبتكر | مدح فارغ | (صف ما هو جديد فعلياً) |
| ثوري | مبالغة | محسّن / مطوّر |
| متعدد الجوانب | حشو | يشمل / يتضمن |
| ركيزة | ج语言 الشركة | أساس / عنصر |
| نموذجي | أكثر من اللازم | ممثّل / قياسي |
| يُمكّن | مبهم | يوفر / يتيح / يعضّد |

### تعبيرات محظورة (احذف بالكامل)

**عبارات التمهيد الفارغ:**
- "من المهم الإشارة إلى أن..."
- "في المشهد الصناعي الحالي..."
- "لن深入了解..."
- "بدون تأخير..."
- "في هذا المقال، سنستعرض..."
- "كما هو معروف..."

**تحفظات فارغة:**
- "لكون الصادق..."
- "في النهاية..."
- "في ما يتعلق بـ..."
- "من ناحية..."
- "بخصوص..."

**حماس الذكاء الاصطناعي:**
- "إنها خطوة كبيرة"
- "...وهو أمر إيجابي!"
- "ها هو الموضوع:"
- "...وهذا جيد!"

### عبارات محددة لسياق الصناعة

**تجنب:** "يمثل هذا تحولاً جذرياً في القطاع الصناعي المغربي"
**تجنب:** "إنها نقطة تحول في تاريخ الصناعة المغربية"
**قل بدلاً من ذلك:** صِف التغيير الفعلي بدون مبالغة

### اختصارات يجب تجنبها في النص العربي

- لا تستخدم اختصارات إنجليزية في النص العربي بدون شرح
- اكتب "الوكالة الوطنية لترويج الصناعة والتجارة" بدلاً من "ANPIMC"
- اذكر الاختصار بين قوسين في أول استخدام فقط

## ENGLISH — Banned Words and Phrases

### Banned Words (Delete or Replace)

| Word | Why it's bad | Alternative |
|------|--------------|-------------|
| delve | AI signature word | explore, examine, look at |
| realm | Pretentious | area, field |
| tapestry | Purple prose | (delete entirely) |
| landscape | Corporate speak | situation, environment |
| leverage | Jargon | use |
| utilize | Jargon | use |
| robust | Meaningless | strong, reliable |
| seamless | Almost always false | smooth, easy |
| comprehensive | Rarely necessary | full, complete |
| cutting-edge | Cliché | modern, new |
| holistic | Corporate | complete, full |
| synergy | Corporate | (delete, rewrite) |
| paradigm | Overused | model, approach |
| empower | Vague | (be specific about what capability) |
| innovative | Empty praise | (describe what's new) |
| transformative | Hyperbolic | changed, improved |
| sophisticated | Vague filler | advanced, or describe what's complex |
| leveraging | Corporate jargon | using |
| ecosystem | Overused metaphor | system, environment |
| rich | (as modifier) | (delete or be specific) |
| over (for quantity) | AP rule | more than |

### Banned Phrases

**Throat-clearing (delete entirely):**
- "It's important to note that..."
- "In today's [X] landscape..."
- "Let's dive/delve into..."
- "Without further ado..."
- "In this article, we will..."
- "It's worth mentioning that..."
- "It goes without saying..."
- "As we all know..."

**Empty hedges:**
- "To be fair..." / "To be honest..."
- "At the end of the day..."
- "When it comes to..."
- "In terms of..."
- "With respect to..."

**AI enthusiasm:**
- "This is a game-changer"
- "...and that's a good thing!"
- "Here's the thing:"
- "...and that's okay!"
- "...and I'm here for it"

**Corporate buzzwords:**
- "Moving forward..."
- "Going forward..."
- "At this point in time..."
- "Due to the fact that..."
- "In order to..."

### Banned Structures

**Don't start sentences with:**
- "So," or "Well,"
- "Now," (when not about time)
- "Look," or "Listen,"
- "Basically," or "Essentially,"

**Don't end sentences with:**
- "...right?"
- "...you know?"
- "...if you will"

### The Substitution Table

| If you wrote | Write instead |
|--------------|---------------|
| utilize | use |
| facilitate | help |
| implement | build, add, create |
| leverage | use |
| functionality | feature |
| methodology | method |
| in order to | to |
| due to the fact that | because |
| at this point in time | now |
| a large number of | many |
| in the event that | if |
| prior to | before |
| subsequent to | after |
| in close proximity to | near |
| has the ability to | can |

### Case Sensitivity

**Always use sentence case for headings, not title case.**

| Wrong | Right |
|-------|-------|
| Morocco's Automotive Sector Reaches New Heights | Morocco's automotive sector reaches new heights |
| The Future Of Renewable Energy In Morocco | The future of renewable energy in Morocco |

## Industrial Journalism Patterns to Avoid (All Languages)

### "Not just X — it's Y" Pattern

**Avoid (FR):** "Ce n'est pas seulement une usine — c'est un symbole"
**Avoid (AR):** "ليست مجرد مصنع — إنها رمز"
**Avoid (EN):** "This isn't just a factory — it's a symbol"

**Do instead:** State the thing directly without dramatic framing

### "Fundamentally Transforms" Pattern

**Avoid (FR):** "Cela transforme fondamentalement l'industrie"
**Avoid (AR):** "يُحدث تحولاً جذرياً في الصناعة"
**Avoid (EN):** "This fundamentally transforms the industry"

**Do instead:** Describe the actual change without hyperbole

### Inflated Claims in Industrial Reporting

**Avoid:** "Une révélation bomb"
**Avoid:** "خطوة استثنائية"
**Avoid:** "A groundbreaking development"

**Do instead:** Let the facts speak. Readers judge significance.

### Empty Transitions

**Avoid (FR):** "En tenant compte de ce qui précède..."
**Avoid (AR):** "بناءً على ما سبق..."
**Avoid (EN):** "Building on this foundation..."

**Do instead:** Just make the next point

### "The Reality Is..." / "La vérité c'est que..." / "الحقيقة أن..."

**Avoid:** All three in any language.
**Do instead:** Just state the claim. The opener adds words and signals AI-generated framing.

### Tricolon Abuse (Reflexive Three-Item Lists)

**Avoid (FR):** "Précis, complet et convaincant"
**Avoid (AR):** "دقيق وشامل ومقنع"
**Avoid (EN):** "Precise, comprehensive, and compelling"

**Do instead:** Use one or two specific points. Three-item parallel lists are an AI rhythm pattern.

## Before and After Examples

### Example 1: French Industrial Report

**AI slop:**
> Dans le paysage industriel marocain en constante évolution, il est crucial de comprendre la nature multidimensionnelle des défis auxquels fait face le secteur automobile. Plongeons dans les solutions robustes et innovantes qui permettent aux constructeurs de tirer parti des opportunités transformationnelles.

**IDM writing:**
> Le secteur automobile marocain a produit 560 000 véhicules au premier semestre 2026, en hausse de 8 % par rapport à 2025. La progression vient mainly des exportations vers l'Europe.

### Example 2: Arabic Investment Announcement

**AI slop:**
> يهدف هذا المشروع الرائد إلى تعزيز المشهد الصناعي المغربي من خلال تقديم حلول مبتكرة وشاملة تُمكّن المقاولات من تحقيق أداء استثنائي في بيئة تنافسية متطورة.

**IDM writing:**
> المشروع سيكلّف 1,2 مليار درهم وسيوفر 850 وظيفة مباشرة عند بدء التشغيل في الربع الثالث من 2027، وفقاً للوكالة الوطنية لترويج الصناعة والتجارة.

### Example 3: English Factory Profile

**AI slop:**
> This isn't just a factory — it's a testament to Morocco's transformative industrial journey. In the rich tapestry of North African manufacturing, few facilities are as compelling as the one about to unfold.

**IDM writing:**
> The Tangier Automotive City plant opened in March 2026. It can produce 200,000 engines per year and employs 1,200 workers.

### Example 4: Newsletter Excerpt

**AI slop:**
> Moving forward, stakeholders in the Moroccan industrial ecosystem must leverage innovative approaches to navigate the challenging landscape of digital transformation. At the end of the day, the companies that embrace this paradigm shift will be the ones that thrive.

**IDM writing:**
> Three industrial zones plan to install fiber-optic networks by 2027. The project targets Tangier, Casablanca, and Kenitra.

## The Verbal Tic Test (All Languages)

Read your text aloud. Does it sound like:
- A TED talk introduction? **Rewrite.**
- A LinkedIn post? **Rewrite.**
- A press release? **Rewrite.**
- Corporate communications? **Rewrite.**
- How you'd explain it to a colleague? **Keep it.**

## Quick Self-Check Before Publishing

Search your text for:

**French:**
1. "approfondir" → examinez, analysez
2. "paysage" → situation
3. "crucial" → est-il vraiment crucial?
4. "robustes" → que signifie ici?
5. "levier" → utilisez
6. "complet" → fait-il un travail?
7. Commencer par "Alors," → supprimez

**Arabic:**
1. "يشمل" → كم عنصر يشمل فعلياً؟
2. "مبتكر" → صِف ما هو جديد
3. "شامل" → هل هو فعلاً كامل؟
4. "يهدف إلى" → حدد الهدف
5. "يُمكّن" → يوفر / يتيح

**English:**
1. "delve" → delete or replace
2. "landscape" → delete or replace
3. "crucial" → is it actually crucial?
4. "robust" → what does this mean here?
5. "leverage" → just say "use"
6. "comprehensive" → is this word doing any work?
7. Starting "So," → delete

Found any? Your writing needs another pass.

## Red Flags in Technical/Industrial Writing

1. **Lists of near-synonyms:** "comprehensive, sophisticated, and robust" (pick one or none)
2. **Excessive hedging:** "may potentially be able to possibly..."
3. **Noun stacking:** "production-ready deployment system infrastructure"
4. **Passive voice hiding agency:** "It was determined that..." (by whom?)
5. **Circular definitions:** "The system enables users to use the functionality"

## The Ultimate Test (All Languages)

Before accepting any AI-assisted text, ask:

1. Can I delete this word/phrase without losing meaning? → Delete it
2. Is this the simplest way to say this? → Simplify
3. Would I say this out loud to a colleague? → If not, rewrite
4. Does this add information or just sound impressive? → If the latter, cut it
5. Is this how an industrial executive would speak? → If not, rewrite
