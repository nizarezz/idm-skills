# IDM Pre-Publication Review

This is the final checkpoint before any content publishes. Every article must pass this review. No exceptions.

## When to Use This Skill

- Before publishing any news article, feature, or analysis
- Before sending any newsletter
- Before posting social media content from IDM channels
- Before publishing any sponsored content (additional steps)
- Before distributing any press release

## The Three-Layer Review

Every article passes through three review layers:

```
Layer 1: Factual Accuracy → Layer 2: Style & Voice → Layer 3: Editorial Standards
```

If any layer fails, the article goes back for revision.

---

## Layer 1: Factual Accuracy

### Claim Verification

```markdown
## Factual Accuracy Review

**Article:** [Title]
**Author:** [Name]
**Reviewer:** [Name]
**Date:** [Date]

### All Claims Verified
- [ ] Every statistic has a source and date
- [ ] Every quote is attributed with full identification
- [ ] Every claim has supporting evidence
- [ ] All numbers are correct (check math)
- [ ] All dates are correct
- [ ] All names are spelled correctly (IDM-TERMINOLOGY.md)
- [ ] All titles are current and accurate

### Company Names
- [ ] All company names use official registered names
- [ ] All company names are consistent throughout
- [ ] Acronyms are spelled out on first reference
- [ ] Legal forms are correct (SA, SARL, etc.)

### Government Names
- [ ] All ministry names use official trilingual versions
- [ ] All agency names use official trilingual versions
- [ ] All institutional names are consistent

### Data Verification
- [ ] All statistics cross-referenced with primary sources
- [ ] All investment figures verified (committed vs. disbursed)
- [ ] All employment figures verified (direct vs. indirect)
- [ ] All production figures verified with company/government data
- [ ] Currency correctly stated (DH, EUR, USD)
- [ ] Units correct (tons, kW, vehicles, etc.)

### Source Documentation
- [ ] All sources listed with access dates
- [ ] All interviews documented with dates
- [ ] All data sources cited with time periods
- [ ] All archived URLs documented
```

### Right of Response

```markdown
## Right of Response Verification

### Subjects of Criticism
- [ ] All parties being criticized have been contacted
- [ ] Adequate time given for response (24-48 hours minimum)
- [ ] Responses documented
- [ ] Non-responses documented
- [ ] Response included or noted in article

### Accuracy Challenges
- [ ] Any disputed facts have been re-verified
- [ ] Disputed claims are presented fairly
- [ ] Both sides of any dispute are represented
```

## Layer 2: Style & Voice

### IDM Style Compliance

```markdown
## Style Review

### Headline
- [ ] Sentence case (not title case)
- [ ] Under 12 words
- [ ] No exclamation marks
- [ ] Present tense for past events
- [ ] Specific and informative

### Lede
- [ ] Under 35 words
- [ ] Answers: Who, What, When, Where, Why
- [ ] Includes at least one specific number
- [ ] No editorializing
- [ ] No throat-clearing phrases

### Numbers and Data
- [ ] Follows language-specific number rules (IDM newsroom style)
- [ ] Percentages formatted correctly (5 % in French, 5% in English)
- [ ] Currency formatted correctly (DH/درهم/MAD)
- [ ] Units follow SI conventions
- [ ] No starting sentences with numerals

### Attribution
- [ ] Uses approved attribution verbs (said/a déclaré/قال)
- [ ] Attribution placed after quote (or at natural pause)
- [ ] Full identification on first reference
- [ ] Consistent attribution style throughout

### Paragraphs
- [ ] All paragraphs 1-3 sentences
- [ ] One idea per paragraph
- [ ] No wall of text paragraphs

### Quotation Marks
- [ ] French: guillemets « »
- [ ] Arabic: « » or " "
- [ ] English: " "
- [ ] Correct placement of periods/commas

### Dates and Times
- [ ] Correct format for language
- [ ] Consistent throughout
- [ ] No ambiguous dates ("last Tuesday" without date)

### Abbreviations
- [ ] Spelled out on first reference
- [ ] Used consistently after first reference
- [ ] No periods in most abbreviations (FBI, CIA)
```

### AI Writing Detection

```markdown
## AI Writing Check

Search the article for ALL of the following:

### French
- [ ] "approfondir" → replace with examiner, analyser
- [ ] "paysage" → replace with situation, environnement
- [ ] "crucial" → is it actually crucial?
- [ ] "robuste" → replace with fiable, solide
- [ ] "levier" → replace with utiliser
- [ ] "complet" → is this word doing work?
- [ ] Starting with "Alors," → delete
- [ ] "Il est important de noter" → delete entire phrase

### Arabic
- [ ] "يشمل" → be specific about what it includes
- [ ] "مبتكر" → describe what is actually new
- [ ] "شامل" → is it actually complete?
- [ ] "يهدف إلى" → specify the goal precisely
- [ ] "يُمكّن" → replace with يوفر / يتيح

### English
- [ ] "delve" → delete or replace
- [ ] "landscape" → delete or replace
- [ ] "crucial" → is it actually crucial?
- [ ] "robust" → what does this mean here?
- [ ] "leverage" → just say "use"
- [ ] "comprehensive" → is this word doing work?
- [ ] Starting "So," → delete
- [ ] "It's important to note" → delete entire phrase

### Universal Patterns
- [ ] No "Not just X — it's Y" pattern
- [ ] No "Fundamentally transforms" pattern
- [ ] No "The reality is..." / "La vérité c'est que..." / "الحقيقة أن..."
- [ ] No reflexive three-item lists (unless genuinely three items)
- [ ] No excessive em-dashes between independent clauses
- [ ] No "game-changer" or equivalent in any language
```

## Layer 3: Editorial Standards

### Editorial Independence

```markdown
## Editorial Independence Check

### Sponsored Content
- [ ] If sponsored, clearly labeled in all three languages
- [ ] Sponsored content does not appear in editorial sections
- [ ] Sponsored content is separated from editorial content
- [ ] No editorial influence from advertisers

### Conflicts of Interest
- [ ] Author has no financial interest in subject
- [ ] Author has no personal relationship with subjects
- [ ] No undisclosed conflicts of interest
- [ ] Any potential conflicts disclosed

### Political Neutrality
- [ ] Article does not endorse any political party
- [ ] Article presents multiple perspectives where relevant
- [ ] No political advocacy in news copy
- [ ] Analysis is balanced and evidence-based
```

### Legal Review

```markdown
## Legal Review (if applicable)

### Defamation Risk
- [ ] All claims supported by evidence
- [ ] No unsubstantiated allegations
- [ ] Right of response documented
- [ ] Language is factual, not inflammatory

### Privacy
- [ ] No unnecessary personal information
- [ ] Privacy of non-public figures protected
- [ ] minors' identities protected

### Intellectual Property
- [ ] No unauthorized use of copyrighted material
- [ ] Images properly licensed or credited
- [ ] Quotations properly attributed
- [ ] No plagiarism

### Regulatory Compliance
- [ ] No violation of press code
- [ ] No violation of industry-specific regulations
- [ ] No national security issues
```

### Trilingual Consistency

```markdown
## Trilingual Consistency Check

### If article is in multiple languages:
- [ ] All three versions convey the same information
- [ ] Key data points are consistent across versions
- [ ] Quotes are equivalent (not paraphrased differently)
- [ ] Headlines are equivalent in meaning
- [ ] Company/government names are consistent

### If article is in one language:
- [ ] Key terms match IDM-TERMINOLOGY.md
- [ ] No untranslated ministry/company names
- [ ] Technical terms are standard
- [ ] Would make sense if translated
```

## Common Issues and Fixes

### Issue: Unsourced Statistic
**Fix:** Find source, add attribution, or delete statistic

### Issue: Company Name Not Official
**Fix:** Look up in IDM-TERMINOLOGY.md, use official registered name

### Issue: Ministry Name Not Official
**Fix:** Use trilingual version from IDM-TERMINOLOGY.md

### Issue: AI Writing Pattern Detected
**Fix:** Apply idm-ai-writing-detox, rewrite affected sentences

### Issue: No Right of Response
**Fix:** Contact subject, give 24-48 hours, document response or non-response

### Issue: Paragraph Too Long
**Fix:** Break into 2-3 shorter paragraphs

### Issue: Lede Too Long
**Fix:** Cut to 35 words, focus on most newsworthy element

### Issue: Attribution Vague
**Fix:** Add full name, title, organization, date
