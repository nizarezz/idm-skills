# IDM Newsroom Style Guide

Write like a professional industrial journalist. Follow IDM conventions. Keep it tight.

## When to Use This Skill

- Writing news articles, briefs, or headlines in any language
- Editing drafts for publication
- Converting interview notes into publishable copy
- Writing press releases or media advisories
- Creating social media posts for IDM channels

## Language-Specific Style Rules

### French Style (AP + French Conventions)

#### Numbers

| Rule | Example |
|------|---------|
| Spell out one through neuf | "trois témoins" not "3 témoins" |
| Use numerals for 10 and above | "15 personnes assistaient" |
| Always use numerals for ages | "une fillette de 5 ans" |
| Always use numerals for percentages | "5 %" (space before %) |
| Always use numerals for addresses | "123 rue Mohammed V" |
| Always use numerals for money | "5 millions DH" or "5 M DH" |
| Spell out premiers for ordinals | "première place" but "10e anniversaire" |

**Exception:** Never start a sentence with a numeral. Rewrite or spell out.

#### Dates and Times

**Months:**
- Never abbreviate French months
- Use: janvier, février, mars, avril, mai, juin, juillet, août, septembre, octobre, novembre, décembre

**Times:**
- Use 24-hour format: "9h" not "9h00" unless precision requires it
- Use: "9h à 17h" for ranges
- Use "midi" and "minuit" not "12h" and "0h"

**Days:**
- Capitalize days of the week: lundi, mardi, mercredi, jeudi, vendredi, samedi, dimanche
- Don't abbreviate

#### Titles and Names

| Rule | Example |
|------|---------|
| Capitalize formal titles before names | "Le ministre de l'Industrie, Mr. X, a déclaré..." |
| Lowercase titles standing alone | "Le ministre a ajouté..." |
| Use "M." not "Mr." | "M. Mohammed VI" |
| Use "Mme" for women | "Mme Fatima Zahra" |

#### Attribution

| Do | Don't |
|----|-------|
| "a déclaré" | "a affirmé," "a remarqué," "a noté" |
| "a ajouté" | "a souligné," "a précisé" |
| Attribution after quote | Attribution before quote |

**Example:**
> "Nous visons 10 GW d'ici 2030," a déclaré le ministre de l'Industrie.

#### Quotation Marks

- Use guillemets « » in French
- Periods and commas go inside guillemets in French
- Use "guillemets français" not "quotes" for inline quotations

#### Abbreviations

- Spell out on first reference, then abbreviate
- Use periods in French abbreviations: M. OH, etc.
- Company names: use official registered names from IDM-TERMINOLOGY.md

### Arabic Style (Modern Standard Arabic)

#### Numbers

| Rule | Example |
|------|---------|
| Use Eastern Arabic numerals (٠١٢٣٤٥٦٧٨٩) | "٥٠٠ وحدة" |
| Or use Western Arabic numerals (0123456789) | "500 وحدة" |
| Be consistent within one article | Choose one system |
| Thousands separator: ٬ (Arabic comma) | "١٬٠٠٠" |

**IDM recommendation:** Use Western Arabic numerals (0-9) for consistency with international standards.

#### Dates

| Rule | Example |
|------|---------|
| Day-Month-Year format | "13 يوليو 2026" |
| Use Arabic month names | يناير، فبراير، مارس، أبريل، مايو، يونيو، يوليو، أغسطس، سبتمبر، أكتوبر، نوفمبر، ديسمبر |
| No commas in dates | "13 يوليو 2026" not "13 يوليو، 2026" |

#### Titles

| Rule | Example |
|------|---------|
| Use official titles | "وزير الصناعة والتجارة" |
| Don't abbreviate ministerial titles | Write full title at least once |
| Use "السيد" for men, "السيدة" for women | "السيد محمد" |

#### Quotation Marks

- Use « » (guillemets) in Arabic
- Use " " (double quotes) for inline quotations
- Periods and commas go inside quotation marks

#### Arabic Grammar Notes

- Use Modern Standard Arabic (فصحى), not dialect
- Active voice preferred over passive
- Avoid unnecessary English or French loanwords when Arabic equivalent exists
- Technical terms from IDM-TERMINOLOGY.md are the exception

### English Style (AP Style)

#### Numbers

| Rule | Example |
|------|---------|
| Spell out one through nine | "three witnesses" not "3 witnesses" |
| Use numerals for 10 and above | "15 people attended" |
| Always use numerals for ages | "a 5-year-old girl" |
| Always use numerals for percentages | "5%" |
| Always use numerals for addresses | "123 Main St." |
| Always use numerals for money | "$5 million" not "five million dollars" |
| Spell out first through ninth for ordinals | "first place" but "10th anniversary" |

**Exception:** Never start a sentence with a numeral. Rewrite or spell out.

#### Dates and Times

**Months:**
- Abbreviate: Jan., Feb., Aug., Sept., Oct., Nov., Dec.
- Don't abbreviate: March, April, May, June, July
- Only abbreviate with specific dates: "Sept. 15" but "September 2025"

**Times:**
- Use figures with a.m./p.m.: "9 a.m." not "9:00 a.m."
- Noon and midnight: Use the words, not "12 p.m." or "12 a.m."
- Time ranges: "9 a.m. to 5 p.m." or "9-11 a.m."

#### Titles and Names

| Rule | Example |
|------|---------|
| Capitalize formal titles before names | "Mayor Jane Smith said..." |
| Lowercase titles after names | "Jane Smith, the mayor, said..." |
| No courtesy titles on second reference | First: "Jane Smith." Second: "Smith" |
| Use full name on first reference | "Jane Smith" not "Smith" or "Ms. Smith" |

#### Attribution

| Do | Don't |
|----|-------|
| "said" | "stated," "remarked," "noted," "expressed" |
| Attribution after quote | Attribution before quote |
| At first natural pause | At awkward break |

**Examples:**

**Bad:** Mayor Smith stated, "We are committed to this project."

**Good:** "We are committed to this project," Mayor Smith said.

#### Quotation Marks

- Periods and commas always go inside
- Colons and semicolons always go outside
- Question marks and exclamation points: inside if part of quote, outside if not

#### Abbreviations

- Spell out on first reference, then abbreviate
- Don't use periods in most abbreviations: FBI, CIA, NATO
- State abbreviations: Use postal codes (NY, CA) only with full addresses; otherwise use AP abbreviations or spell out

## Trilingual Formatting Rules

### Company Names

**Rule:** Never translate company names. Use the official registered name.

| Language | Example |
|----------|---------|
| French | « Renault Maroc » |
| Arabic | "رينو المغرب" |
| English | "Renault Morocco" |

### Government Ministry Names

**Rule:** Use official trilingual versions from IDM-TERMINOLOGY.md.

| French | Arabic | English |
|--------|--------|---------|
| Ministère de l'Industrie et du Commerce | وزارة الصناعة والتجارة | Ministry of Industry and Commerce |

### Technical Terms

**Rule:** Use the term from IDM-TERMINOLOGY.md. When a term has no direct equivalent, use the closest acceptable translation and note it on first reference.

### Currency

| Language | Format | Example |
|----------|--------|---------|
| French | [amount] DH | 100 DH |
| Arabic | [amount] درهم | 100 درهم |
| English | [amount] MAD | 100 MAD |

### Units

Follow SI conventions in all languages. Use French abbreviations in French text, Arabic in Arabic text, English abbreviations in English text.

## Headline Conventions

### French Headlines
- Sentence case (not title case)
- Present tense for past events: "Renault produit sa millionième voiture"
- No exclamation marks
- Maximum 12 words
- No periods at the end
- Use guillemets for quotes in headlines

### Arabic Headlines
- No punctuation at end
- Active voice preferred
- Maximum 10 words
- Use Modern Standard Arabic
- No exclamation marks

### English Headlines
- Sentence case
- Present tense for past events: "Morocco's automotive sector reaches new export record"
- No exclamation marks
- Maximum 10 words
- No periods at the end

## Article Structures

### News Article (Inverted Pyramid)

1. **Lede:** Who, what, when, where, why (35 words max)
2. **Key facts and figures**
3. **Context and background**
4. **Quotes from officials/executives**
5. **Impact and implications**
6. **Background information**

**Example lede (French):**
> Le secteur automobile marocain a produit 560 000 véhicules au premier semestre 2026, en hausse de 8 %, selon l'AMIF.

**Example lede (Arabic):**
> أproduced قطاع السيارات المغربي 560 ألف وحدة في النصف الأول من 2026، بزيادة 8%، وفقاً لـ AMIF.

**Example lede (English):**
> Morocco's automotive sector produced 560,000 vehicles in the first half of 2026, up 8%, according to AMIF.

### Feature Article

1. Hook (anecdote, scene, question)
2. Context setting
3. Main narrative with data points
4. Expert analysis
5. Industry implications
6. Forward-looking conclusion

### Investigative Article

1. Finding or revelation
2. Evidence presentation
3. Official responses
4. Historical context
5. Impact analysis
6. Accountability conclusion

### Data/Analysis Article

1. Key finding
2. Data source and methodology
3. Detailed analysis with charts
4. Expert interpretation
5. Comparison with benchmarks
6. Implications and outlook

## Paragraph and Quote Rules

### Paragraphs
- Keep paragraphs short: 1-3 sentences
- One idea per paragraph
- Break up long quotes into multiple paragraphs

### Quotes

**French:**
- Use guillemets « »
- Attribution after quote with "a déclaré"
- Period inside guillemets

**Arabic:**
- Use « » or " " 
- Attribution after quote
- Period inside quotation marks

**English:**
- Use " "
- Attribution after quote with "said"
- Period inside quotation marks

## Common Mistakes to Avoid

### All Languages

1. **Unsourced statistics** — every number needs a source and date
2. **Passive voice hiding agency** — "It was decided that..." (by whom?)
3. **Starting sentences with "There is" or "There are"** — restructure
4. **Using "very" or "extremely"** — delete or find a stronger word
5. **Exclamation points in news copy** — never
6. **First-person pronouns** (unless first-person piece)
7. **Unattributed opinions** — always attribute
8. **Editorializing in news copy** — state facts, let readers judge

### French-Specific

1. **Anglicisms:** "marketing," "management," "business" — use French equivalents when available
2. **Inconsistent guillemets:** « » not " " or ' '
3. **Missing spaces before punctuation:** "fait," not "fait,"

### Arabic-Specific

1. **Dialect instead of فصحى:** Always use Modern Standard Arabic
2. **Unnecessary transliteration:** Use Arabic terms from IDM-TERMINOLOGY.md
3. **Missing tashkeel (diacritics):** Add when needed for clarity

### English-Specific

1. **Title case in headlines:** Use sentence case
2. **Oxford comma:** AP doesn't use it; IDM uses it for clarity
3. **"Over" vs. "more than":** Use "more than" for quantities

## Quick Reference Card

### Before You Publish (Any Language)

- [ ] Names spelled correctly (use IDM-TERMINOLOGY.md)
- [ ] Titles correct and styled properly
- [ ] Numbers follow language-specific rules
- [ ] Attribution uses approved verbs (said/a déclaré/قال)
- [ ] Dates and times formatted correctly
- [ ] Paragraphs are short (1-3 sentences)
- [ ] Lede is under 35 words
- [ ] No editorializing in news copy
- [ ] Sources are credible and named
- [ ] Company names use official registered names
- [ ] Government names use official trilingual versions
- [ ] Currency formatted correctly (DH/درهم/MAD)
- [ ] Units follow SI conventions

### Red Flags

- "Very" or "extremely" in news copy
- Exclamation points
- First-person pronouns (unless first-person piece)
- Unattributed opinions
- Passive voice hiding who did what
- Starting sentences with "There is" or "There are"
- Unsourced statistics
- Company names translated from original
