# IDM Industrial Terminology Glossary

## How to Use This Glossary

This glossary provides standardized trilingual terminology for Industrie du Maroc Magazine. All editorial staff must use these terms consistently across French, Arabic, and English content.

**Rules:**
1. Use the exact term in each language — do not improvise alternatives
2. Company names are never translated — use the official registered name
3. Government ministry names follow official trilingual versions
4. Acronyms are listed after full name in first reference
5. When a term has no direct equivalent, the closest acceptable translation is provided
